Formatting tags in Tabular Format
Tags Definition
Bold The Bold tag is an HTML element used to apply bold formatting to text, making it visually stand out by increasing its weight or thickness.
Italic The Italic tag is an HTML element used to apply italic formatting to text, giving it a slanted appearance for emphasis or to indicate a different style or tone.
Underline The Underline tag is an HTML element used to apply an underline to text, creating a horizontal line beneath the text to indicate emphasis, hyperlink, or other visual distinction.
Strong Tag The strong tag is an HTML element used to indicate that its enclosed text is of strong importance, typically rendered in bold to highlight its significance within the document's context.
Small Tag The small tag is an HTML element used to indicate that its enclosed text is of smaller size, typically used for fine print, disclaimers, or secondary information within the document's content.
Big Tag The big tag is an HTML element used to indicate that its enclosed text should be displayed with a larger font size, making it visually stand out and appear more prominent.
Mark Tag The mark tag is an HTML element used to highlight or mark specific portions of text, typically by applying a background color or other visual effect to draw attention to the marked content.
Emphasized Tag The em tag is an HTML element used to indicate that its enclosed text should be emphasized or stressed, typically rendered in italics to convey a different level of importance or significance.
Deleted Tag The del tag is an HTML element used to indicate that its enclosed text has been deleted or removed from the document, typically displayed with a strikethrough line to indicate the deleted content.
Inserted Tag The ins tag is an HTML element used to indicate that its enclosed text has been inserted or added to the document, typically displayed with an underline or other visual effect to indicate the inserted content.
Subscripted Tag SubThe tag is an HTML element used to display text as subscript, positioning it below the baseline of the surrounding text, often used for chemical formulas, mathematical equations, or footnotes.
Superscripted Tag SupThe tag is an HTML element used to display text as superscript, positioning it above the baseline of the surrounding text, often used for mathematical exponents, footnotes, or to denote citations and references.
